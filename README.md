# Valentijn Website

Romantische interactieve Valentijnswebsite met animaties, knoppenlogica en bloemen-thema.

## Inhoud
- Interactieve flow met meerdere schermen
- Romantisch design met zachte kleuren
- Valentijnsvraag met verrassingsanimatie
- Responsief ontwerp voor mobiel en desktop
